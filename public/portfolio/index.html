🎯 프로젝트 주요 기여#

  • Spring Batch 모듈을 통한 데이터 처리 파이프라인 설계
  • QueryDSL을 이용한 동적 검색 쿼리 작성으로 복합 조건 검색 기능 구현
  • Redis를 활용한 세션 관리
