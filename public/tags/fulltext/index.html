프로젝트 상황에 맞는 검색 기술 도입 및 성능 비교 분석 🔍 문제 상황 검색 기능 요구사항 KernelEngine은 수집된 기술 블로그 게시글에서 사용자가 원하는 정보를 빠르게 찾을 수 있는 검색 기능이 핵심이었습니다. 처음엔 단순하게 생각하여 LIKE 검색을 사용하려고 했지만, 검색 성능이 그다지 좋지 않다는 점을 테스트 와중 발견하였습니다. 기술적 고려사항 검색 기능 구현을 위해 다음과 같은 옵션들을 검토했습니다: 1. MySQL LIKE 검색 1 쿼리 예시 추가 필요 장점: 구현이 간단, 추가 인프라 불필요 단점: 성능이 느림, 복잡한 검색 불가능 2. MySQL FULLTEXT 검색 1 쿼리 예시 추가 필요 장점: 내장된 전문 검색 기능, 관련도 점수 제공 단점: 기능이 제한적, 고급 검색 기능 부족 3.
