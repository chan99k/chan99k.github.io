# chan99k's blog

## 📁 프로젝트 구조

```
├── src/
│   ├── app/                 # Next.js app router 페이지
│   ├── components/          # React 컴포넌트
│   ├── lib/                 # 유틸리티 함수 및 설정
│   ├── types/               # TypeScript 타입 정의
│   ├── hooks/               # 커스텀 React 훅
│   ├── utils/               # 헬퍼 유틸리티
│   └── styles/              # 추가 스타일
├── content/                # 콘텐츠 파일
│   ├── portfolio/          # 포트폴리오 데이터
│   ├── blog/               # 블로그 포스트 (MDX)
│   └── reviews/            # 맛집 리뷰
└── public/                 # 정적 자산
    ├── images/             # 콘텐츠 타입별 이미지
    └── icons/              # 아이콘 자산
```
