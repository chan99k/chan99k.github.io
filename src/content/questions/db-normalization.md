## 해설

정규화 예시:
```
1NF 위반: { id: 1, hobbies: "독서, 영화" }  // 원자값 아님
1NF: { id: 1, hobby: "독서" }, { id: 1, hobby: "영화" }

2NF 위반: { 학생ID, 과목ID, 학생이름, 점수 }  // 학생이름은 학생ID에만 종속
2NF: { 학생ID, 학생이름 }, { 학생ID, 과목ID, 점수 }

3NF 위반: { 직원ID, 부서ID, 부서명 }  // 부서명은 부서ID에 종속
3NF: { 직원ID, 부서ID }, { 부서ID, 부서명 }
```

역정규화 사례: 주문 테이블에 주문 시점의 상품 가격 저장(상품 테이블 조인 없이 조회)
