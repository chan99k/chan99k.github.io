Spring Modulith 기반 Modular Monolith 아키텍처로 설계한 선물 펀딩 플랫폼입니다.

주요 기여:
- Spring Modulith 기반 Modular Monolith 아키텍처 설계 및 회원·결제·지갑·알림 모듈 구현
- k3s(Kubernetes) 배포 환경 구축 및 Prometheus/Grafana 모니터링 구성
- Github Actions CI 파이프라인 구축 (빌드/테스트 자동화, Jacoco 커버리지 리포트)
- Auth0 OAuth2 인증/인가 및 Redis 기반 토큰 블랙리스트 관리
