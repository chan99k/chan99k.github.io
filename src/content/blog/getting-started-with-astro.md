Astro는 콘텐츠 중심 웹사이트를 만들기 위한 최적의 프레임워크입니다.

## 왜 Astro인가?

- **제로 JS 기본**: 클라이언트에 자바스크립트를 보내지 않습니다
- **아일랜드 아키텍처**: 필요한 컴포넌트만 하이드레이션
- **Content Collections**: 타입 안전한 마크다운 관리

## 시작하기

```bash
npm create astro@latest
```

간단한 명령어 하나로 프로젝트를 시작할 수 있습니다.
